# ami-starter
Starter kit for AMI (AMI Medical Imaging)

AMI 0.0.4.

### Getting started

```
$> git clone https://github.com/FNNDSC/ami-starter.git

$> cd ami-starter
$> npm install
```

### Gulp tasks

```
# live reload + bundle all files (watchify + babelify + glslify)
$> gulp serve

# bundle all files (watchify + babelify + glslify)
$> gulp
```
